# Problem

AI agents are rapidly being deployed across engineering, support, finance, and operations teams.

However, most organizations cannot answer:

- Which agents are performing well?
- Which agents are drifting over time?
- Which agents are introducing operational risk?
- Which agents are creating measurable business value?

As AI adoption grows, trust becomes the biggest challenge.

---

# Solution

CORAXL provides a governance layer for autonomous AI agents.

Using Coral SQL, CORAXL transforms raw agent telemetry into:

- Agent Health Scores
- Drift Detection
- Risk Assessment
- Governance Recommendations
- Executive Insights
- Explainable Audit Trails

Every recommendation generated by the platform is traceable through transparent Coral SQL queries.

# Environment Variables

Create a `.env` file inside the backend directory:

```env
GEMINI_KEY=YOUR_GEMINI_API_KEY
```

### Security Note

- Never commit real API keys to GitHub
- Add `.env` to `.gitignore`
- Use `.env.example` as a template

---

# Run Locally

## Frontend

```bash
npm install
npm run dev
```

## Backend

```bash
cd backend

pip install flask
pip install flask-cors
pip install requests

python3 server.py
```

Backend endpoint:

```text
http://127.0.0.1:5000/api/health
```
